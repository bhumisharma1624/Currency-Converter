# Real-Time Currency Converter 💱

A clean, responsive web application that provides real-time currency conversion rates. Built using modern HTML5, CSS3, and Vanilla JavaScript, this project integrates a live exchange rate API to deliver instant, accurate conversions across global currencies.

---

## ✨ Features
* **Live Exchange Rates:** Integrates a third-party REST API to fetch up-to-the-minute exchange rate calculations.
* **Smart Dropdown Selectors:** Fully populated currency selectors covering major global currencies, featuring automated country flag updates upon selection.
* **Instant Conversion Logic:** Automatically calculates and displays the converted value with precise decimal accuracy upon button click or input change.
* **Input Validation:** Prevents negative inputs or invalid characters, ensuring the app handles empty states gracefully by defaulting values.
* **Modern UI/UX:** A minimalist, card-based glassmorphism design that scales seamlessly from desktop displays down to mobile screens.

---

## 🛠️ Tech Stack
* **HTML5:** Structured the form inputs, currency selection menus, and display elements.
* **CSS3:** Custom dropdown styling, Google Fonts typography integration, and responsive layout scaling.
* **JavaScript (ES6+):** Async/Await operations, Fetch API for data retrieval, DOM manipulation, and dynamic flag updates.

## 🧠 Technical Learnings
Developing this application helped me master essential JavaScript data-handling skills:
1.  **Asynchronous JavaScript (Promises & Fetch):** Learned to securely query a public API, handle server responses asynchronously, and catch network errors seamlessly.
2.  **Working with JSON Data:** Mastered parsing complex nested JSON objects returned by foreign exchange APIs to extract specific currency values dynamically.
3.  **Event Handling:** Applied event listeners to track user keypresses and dropdown modifications, ensuring a responsive user experience.
